# How to Enable GitHub Pages

Follow these step-by-step instructions to enable GitHub Pages for your repository and obtain your live link:

## Step 1: Navigate to Your Repository
- Go to your GitHub profile.
- Click on the `Repositories` tab.
- Select the repository you want to use for GitHub Pages (e.g., `ug-pg`).

## Step 2: Access Settings
- Click on the `Settings` tab located on the right side of the repository.

## Step 3: Scroll to GitHub Pages Section
- Scroll down to the `GitHub Pages` section. It might be under the `Code and automation` menu.

## Step 4: Choose the Source
- In the `Source` dropdown menu, choose the branch you want to use for GitHub Pages, typically `main` or `gh-pages`.
- Optionally, you can choose a folder like `/ (root)` or `/docs`.

## Step 5: Save Settings
- Click `Save` to apply the changes.

## Step 6: Verify Publication
- After a few moments, GitHub will display a message confirming that GitHub Pages is enabled.
- You will see the live link to your published site in the same section.

## Step 7: Access Your Live Link
- Copy the provided link and paste it into your web browser to view your GitHub Pages site.

## Additional Tips:
- Make sure your repository is public if you want others to access the GitHub Pages.
- You can customize the theme and settings further under `Settings > Pages`.
